Description

We host a publicly available VerneMQ MQTT broker. MQTT is a very lightweight protocol that uses a publish/subscribe model. This makes it suitable for "machine to machine" messaging such as with low power sensors or mobile devices.

Testing and usage is for free but please do not use it for sensitive information because everybody is allowed to subscribe to every topic, including wildcard.

Please consider to add a reconnect logic to your client because we may update the underlying VerneMQ instance at any time, so we cannot promise 100% uptime.
